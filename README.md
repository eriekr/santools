How to get your key:
1. Get HWID 
   - Fla
     Fla> setting hwid
   - SETTING
     SETTING> hwid
2. Email your hwid 
   to: 
   subject: Register Fla
   Email body: [HWID]
3. After receiving email for key, apply new key
   - Fla
     Fla> setting applykey
   - SETTING
     SETTING> applykey
